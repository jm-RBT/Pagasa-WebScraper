WORKFLOW
================================================================================
1. Place PDFs in dataset/pdfs/ directory
2. Run pdf_annotation_gui.py
3. GUI opens with first PDF loaded
4. Automatic extraction displays JSON
5. Edit JSON if needed
6. Click "Save & Next" to save and proceed
7. Repeat for all PDFs
8. Annotations saved to dataset/pdfs_annotation/
